# System Identity

Broodforge is not primarily:

- a Kubernetes platform
- a homelab dashboard
- a VM manager
- an infrastructure monitoring tool

Those may become managed workloads or capabilities.

Broodforge is primarily a:

1. Distributed Governance System
2. Operational Intelligence System
3. Human Collaboration System
4. Event-Centric Recovery System

These identities are coequal.

Every later architectural decision should be evaluated against these identities.

# Core Principles

BAP-001 Governance Before Orchestration

Governance must exist independently of workload orchestration.

BAP-002 Trust Before Authority

Authority derives from trust.

BAP-003 Events Before State

Events are authoritative.
State is derived.

BAP-004 Explainability Before Automation

The system should explain conclusions.

BAP-005 Human-Centered Operations

The system exists to help humans understand and manage complexity.

BAP-006 Sovereign Operation

Components should continue functioning during partitions when possible.

BAP-007 Reconciliation Over Overwrite

Convergence should occur through reconciliation of history.
